O Git foi projetado e desenvolvido por Linus Torvalds para o 
desenvolvimento do Linux, mas, como o GIT atendia necessidades
e oferecia mais soluções para o versionamento do que outros softwares
concorrentes, acabou sendo adotado por muitos outros projetos.




b. Quem atualmente é o dono do Github e por quê?


A microsoft. Para cooptar desenvolvedores que utilizam sistemas em código aberto
e com isso atrai-los para usarem outras ferramentas desenvolvidas pela microsoft , aumentanto o número de consumidores.





c. Existe outra maneira, além do terminal, de trabalhar com o Github?

Sim. GitHub Desktop permite um interação com o GitHub usando uma 
GUI em vez da linha de comando ou de um navegador web.
